# Sistema de Taller Mecánico (Proyecto de práctica)

## Descripción
El mini proyecto que he desarrollado consiste en un taller mecánico capaz de registrar y mostrar información sobre los autos que llegan a reparar contando con datos básicos para entender que necesita el cliente tales como problema técnico, cliente, marca,  etc.

## Tecnologías utilizadas
- Lenguaje utilizado python
- Enfoque POO
- Herramientas Git, Github

## Instalación
Para poder ejecutar este codigo se necesita de estas herramientas siguiendo estos paso
1. instalar python
2. instalar visual studio code 
3. instalar la extensión python en visual studio

## Ejecución
```
python Taller.py
```
